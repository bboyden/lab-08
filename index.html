Lab 8: Accessing Photo Data via REST

This lab focused on building a Node.js + Express server that fetches photo data from an external API and serves it through custom routes.

Key Skills Learned:

  • Created a Node.js project and installed Express
  • Fetched and served photo data from JSONPlaceholder API
  • Used fetch(), promises, and chaining in server code
  • Built client-side code with HTML, CSS, and JavaScript to display results
  • Learned how to serve static files from a public directory
